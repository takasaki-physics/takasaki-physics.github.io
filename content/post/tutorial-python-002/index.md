## 目的
文字の出力方法を解説します。
今後、プログラムを動かすときに正常に動作しているかどうかを確認するなど、利用の機会が多いコマンドです。

## Hello World!
`Hello World!` を出力します。

```python
printf ("Hello World!")

# 出力> Hello World!
```

## 変数の出力
ここでは先に数値の出力方法を記します。
(変数の扱い方は別の文章で詳しく解説します。)

```python
int number = 200

printf (number)

# 出力> 200
```

文字式も同様に可能です。

```python
string text = "Apple"

printf (text)

# 出力> Apple
```
