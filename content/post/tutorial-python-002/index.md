## 目的
文字の出力方法を解説します。
今後、プログラムを動かすときに正常に動作しているかどうかを確認するなど、利用の機会が多いコマンドです。

文字の出力では `print()` という関数を利用します。

## Hello World!
### 文字列の出力
`Hello World!` を出力します。

```python
print("Hello World!")

# 出力> Hello World!
```

### 変数の出力
変数を利用した出力方法を紹介します。
(変数の扱い方は別の文章で詳しく解説します。)

数値の出力例:
```python
int number = 200
print(number)

# 出力> 200
```

文字の出力例:
```python
string text = "Apple"
print(text)

# 出力> Apple
```
